# 期货交易入门手册

## 1. 什么是期货

### 1.1 期货的定义

期货（Futures）是一种标准化的金融合约，买卖双方约定在**未来某个特定日期**，以**事先确定的价格**买入或卖出一定数量的标的资产。

简单理解：期货就是"现在约定价格，将来交货"的交易。

与日常买卖不同的是：
- 期货合约是**标准化**的，由交易所统一制定合约规格（品种、数量、质量、交割地点等）。
- 交易不需要全额付款，只需缴纳一定比例的**保证金（Margin）**。
- 合约到期前可以随时买卖转让，不必真的进行实物交割。

### 1.2 期货的起源与发展

期货交易最早可追溯到17世纪日本大阪的"堂岛米会所"，商人们为了规避米价波动风险，开始进行远期稻米合约的交易。

现代期货市场的标志性事件：
- **1848年**：美国芝加哥期货交易所（CBOT）成立，最初交易农产品（小麦、玉米）。
- **1972年**：芝加哥商业交易所（CME）推出外汇期货，标志着金融期货的诞生。
- **1982年**：股指期货出现（标普500指数期货），期货从商品扩展到金融领域。
- **1990年**：中国郑州粮食批发市场引入期货交易机制，中国期货市场起步。
- **2010年**：中国推出沪深300股指期货，金融期货正式登场。

### 1.3 期货的市场价值与功能

期货市场并非单纯的投机场所，它对实体经济有重要的服务功能：

**价格发现（Price Discovery）**

期货市场汇聚了大量的买卖双方，通过公开竞价形成的价格反映了市场对未来供需的预期。这个价格信号被产业链上下游广泛参考，帮助企业做出生产和经营决策。例如，农民可以参考玉米期货价格来决定来年的种植面积。

**套期保值 / 风险转移（Hedging）**

这是期货最核心的功能。生产者和消费者可以通过期货市场锁定未来的买卖价格，将价格波动风险转移给愿意承担风险的投机者。例如：
- 航空公司买入原油期货锁定燃油成本。
- 铜矿企业卖出铜期货锁定销售价格。

**提高市场流动性**

期货市场的标准化合约和保证金制度降低了交易门槛，吸引了大量参与者，使得大宗商品和金融资产可以高效流转，减少了买卖双方的搜寻成本。

**资源配置优化**

期货价格引导资源流向更有效率的领域。当某种商品期货价格上涨时，信号传递到生产端，引导增加供给；价格下跌则引导减少产能，从而促进资源的合理配置。

**稳定经济运行**

通过套期保值，企业能够稳定经营预期，减少因价格剧烈波动导致的经营困难，间接起到稳定经济的作用。

### 1.4 期货与现货的区别

| 对比项 | 现货 | 期货 |
|--------|------|------|
| 交易对象 | 实物商品 | 标准化合约 |
| 交割时间 | 即时或短期 | 约定的未来日期 |
| 交易场所 | 分散、多样 | 交易所集中撮合 |
| 付款方式 | 全额付款 | 保证金交易（通常5%~15%） |
| 目的 | 获取实物 | 套期保值或投机获利 |

### 1.5 期货与股票的区别

| 对比项 | 股票 | 期货 |
|--------|------|------|
| 交易方向 | 只能做多（融券做空门槛高） | 双向交易，做多做空同样方便 |
| 杠杆 | 无杠杆（融资融券除外） | 保证金交易，自带杠杆 |
| 到期日 | 无到期日，可长期持有 | 有到期日，需在到期前平仓或交割 |
| 交易时间 | T+1（A股当日买入次日才能卖出） | T+0（当日可多次开平仓） |
| 持有成本 | 无持仓费用 | 保证金占用，可能追加保证金 |
| 结算方式 | 交易时结算 | 每日结算（逐日盯市） |

## 3. 核心概念

### 3.1 合约要素

每个期货合约都有交易所制定的标准化规格，主要包括：

- **合约标的**：具体的商品或金融资产（如螺纹钢、沪深300指数）。
- **合约单位（Contract Size）**：一手合约代表的标的数量。如螺纹钢 10吨/手，沪深300股指期货 300元/点。
- **最小变动价位（Tick Size）**：价格变动的最小单位。如螺纹钢为1元/吨，即每跳动1个最小单位，一手盈亏 = 1元 × 10吨 = 10元。
- **交割月份**：合约到期的月份，如2026年3月合约记为 RB2603。
- **最后交易日**：合约可以交易的最后一天，个人投资者需在此之前平仓。
- **交割方式**：实物交割或现金交割。

### 3.2 保证金与杠杆

**保证金（Margin）** 是交易期货合约时需要缴纳的履约担保资金，通常为合约价值的 5%~15%。

举例：螺纹钢价格 3500元/吨，合约单位 10吨/手
- 合约价值 = 3500 × 10 = 35,000元
- 保证金比例 10%，则一手保证金 = 35,000 × 10% = 3,500元
- **杠杆倍数** = 合约价值 / 保证金 = 35,000 / 3,500 = **10倍**

杠杆是一把双刃剑：收益和亏损都会被放大。价格变动1%，保证金盈亏变动10%。

### 3.3 做多与做空

- **做多（Long）**：预期价格上涨，先买入开仓，价格上涨后卖出平仓获利。
- **做空（Short）**：预期价格下跌，先卖出开仓，价格下跌后买入平仓获利。

期货的双向交易机制使得无论市场涨跌，交易者都有获利机会。

```
做多流程：买入开仓 → 价格上涨 → 卖出平仓 → 赚取差价
做空流程：卖出开仓 → 价格下跌 → 买入平仓 → 赚取差价
```

### 3.4 开仓、平仓与持仓

- **开仓**：建立新的期货头寸（买入开仓 或 卖出开仓）。
- **持仓**：持有未平仓的合约。
- **平仓**：通过反向交易了结持仓（买入平仓 或 卖出平仓）。
- **平今仓**：对当日开仓的合约进行平仓，部分品种平今仓手续费较高。

### 3.5 结算价与收盘价

- **收盘价**：当日最后一笔成交价格。
- **结算价**：交易所按一定规则计算的当日加权平均价，用于每日结算（逐日盯市）。

> 注意：期货每日盈亏是按**结算价**而非收盘价计算的，两者可能存在差异。

### 3.6 涨跌停板

交易所对每日价格波动设有上下限，超过该限制则触发涨停或跌停。例如，螺纹钢涨跌停幅度通常为上一交易日结算价的 ±4%~7%（具体比例交易所会根据市场情况调整）。

涨跌停后合约仍可交易，但价格不能超过停板价格。连续涨跌停时，交易所可能采取扩板、强制减仓等措施。

### 3.7 交割

**实物交割（Physical Delivery）**：
合约到期时，卖方交付实物商品，买方支付货款。大多数商品期货采用实物交割。个人投资者一般不能参与实物交割，需在最后交易日前平仓。

**现金交割（Cash Settlement）**：
合约到期时按结算价进行现金差额结算，不涉及实物交收。股指期货、国债期货采用现金交割。

> 提示：实际上绝大多数期货合约在到期前就已经被平仓，只有极少数合约进入交割环节。

### 3.8 主力合约与合约换月

同一品种同时有多个月份的合约在交易，其中成交量和持仓量最大的合约称为**主力合约（Most Active Contract）**，流动性最好，是大多数交易者参与的合约。

**换月（Roll Over）**：当主力合约临近到期时，市场活跃度会逐渐转移到下一个月份的合约，这个过程称为换月。交易者需要关注换月时机，避免持有即将到期的不活跃合约。

## 4. 交易流程

### 4.1 开户流程与条件

**基本条件：**
- 年满18周岁，具有完全民事行为能力。
- 持有有效身份证件。
- 具备一定的风险承受能力。

**开户方式：**
1. **线上开户（主流）**：下载期货公司官方APP，按指引提交身份证照片、银行卡信息、进行视频验证。
2. **线下开户**：前往期货公司营业部办理。

**特殊品种开户门槛：**
部分品种有额外要求：
- **股指期货**：需50万元资金验证 + 知识测试（80分以上）+ 10笔以上商品期货交易经验。
- **原油、铁矿石等特定品种**：需10万元资金验证或通过知识测试 + 交易经验。
- **普通商品期货**：无资金门槛。

### 4.2 入金与出金

- **入金**：通过银期转账（银行卡绑定期货账户），从银行账户划转资金到期货账户，实时到账。
- **出金**：从期货账户划转资金到银行卡，一般在交易日 9:00~15:30 可操作，次日到账或实时到账。
- 出金有留底资金要求（通常几百到一千元不等）。

### 4.3 下单方式

| 下单类型 | 说明 |
|----------|------|
| **限价单** | 指定价格，只在该价格或更优价格成交 |
| **市价单** | 以当前最优价格立即成交，可能存在滑点 |
| **止损单** | 价格触及设定值时自动触发下单 |
| **止盈单** | 价格达到目标盈利时自动触发平仓 |
| **条件单** | 满足预设条件时自动下单（由交易软件提供） |

> 建议新手优先使用限价单，避免市价单在行情剧烈波动时出现较大滑点。

### 4.4 一笔完整交易的生命周期

```
开户 → 入金 → 分析行情 → 开仓下单 → 持仓监控 → 平仓了结 → 结算盈亏
```

详细步骤：
1. 选择要交易的品种和合约月份（通常选择主力合约）。
2. 判断方向：看涨则买入开仓（做多），看跌则卖出开仓（做空）。
3. 确定仓位大小：根据账户资金和风险承受能力计算开仓手数。
4. 下单成交后进入持仓状态。
5. 盯盘或设置止损止盈，管理风险。
6. 达到目标或触发止损后平仓。
7. 交易所按结算价进行每日结算，盈亏划入账户。

### 4.5 手续费与交易成本

期货交易成本主要包括：

- **手续费**：分为按手数收取（如螺纹钢约 1~5元/手）和按成交金额比例收取（如股指期货万分之0.23）。
- **平今仓手续费**：部分品种当日平仓手续费较高，甚至为开仓手续费的数倍。
- **滑点成本**：实际成交价与预期价格的偏差。
- 无印花税、无过户费（与股票不同）。

## 5. 盈亏计算

### 5.1 盈亏计算公式

**做多盈亏：**

```
盈亏 = (平仓价 - 开仓价) × 合约单位 × 手数
```

**做空盈亏：**

```
盈亏 = (开仓价 - 平仓价) × 合约单位 × 手数
```

### 5.2 保证金占用与可用资金

```
占用保证金 = 合约价值 × 保证金比例 × 手数
可用资金 = 账户权益 - 占用保证金
```

当可用资金不足时，期货公司会通知追加保证金。

### 5.3 强制平仓（爆仓）机制

当账户可用资金为负且未在规定时间内补足保证金时，期货公司有权对持仓进行**强制平仓**。

触发条件：
- 风险度 = 占用保证金 / 账户权益 ≥ 100%（各期货公司标准略有不同）。
- 收到追保通知后未在次日开盘前补足资金。

强制平仓后的亏损由投资者承担，极端行情下甚至可能出现**穿仓**（亏损超过账户全部资金）。

### 5.4 计算示例

**场景：** 做多螺纹钢 RB2605

| 项目 | 数值 |
|------|------|
| 开仓价 | 3500 元/吨 |
| 合约单位 | 10 吨/手 |
| 开仓手数 | 2 手 |
| 保证金比例 | 10% |
| 平仓价 | 3600 元/吨 |

计算过程：
```
合约价值 = 3500 × 10 × 2 = 70,000 元
占用保证金 = 70,000 × 10% = 7,000 元
盈亏 = (3600 - 3500) × 10 × 2 = 2,000 元（盈利）
收益率 = 2,000 / 7,000 ≈ 28.6%
```

> 注意：价格只上涨了 2.86%，但因为杠杆效应，保证金收益率达到 28.6%。反之如果价格下跌2.86%，亏损同样是28.6%。

## 6. 风险管理

### 6.1 期货交易的主要风险

| 风险类型 | 说明 |
|----------|------|
| **杠杆风险** | 保证金交易放大盈亏，小幅波动即可造成大额亏损 |
| **强平风险** | 保证金不足时被强制平仓，可能在最不利的价格成交 |
| **流动性风险** | 非主力合约或极端行情下可能无法及时平仓 |
| **隔夜风险** | 持仓过夜面临外盘波动、突发事件导致的跳空风险 |
| **交割风险** | 个人投资者未及时平仓可能面临交割违约 |

### 6.2 仓位管理原则

- **单品种仓位不超过总资金的 20%~30%**。
- **总仓位不超过总资金的 50%~60%**。
- 预留足够的可用资金应对价格波动和追保。
- 新手建议从**1~2手**小仓位起步。

### 6.3 止损与止盈

**止损（Stop Loss）**：
- 开仓前**必须**确定止损位，这是最基本的风险控制手段。
- 常见方法：固定金额止损（如单笔亏损不超过总资金的2%）、技术位止损（如跌破支撑位）。

**止盈（Take Profit）**：
- 设定合理的盈利目标，达到后部分或全部平仓。
- 可采用移动止盈（Trailing Stop），随价格上涨逐步上移止损位，锁定利润。

### 6.4 资金管理建议

1. **用闲钱交易**：不要使用借贷资金或生活必需资金。
2. **控制单笔风险**：单笔交易亏损不超过总资金的 1%~3%。
3. **分散品种**：不要把所有资金押注在单一品种上。
4. **及时出金**：盈利达到一定比例后转出部分利润，落袋为安。

## 7. 基本分析方法简介

### 7.1 基本面分析

基本面分析关注影响商品供需关系和价格的根本因素：

- **供给端**：产量、库存、产能、进出口政策。
- **需求端**：下游消费、开工率、季节性需求变化。
- **宏观经济**：利率、汇率、通胀、货币政策。
- **政策因素**：环保限产、关税调整、补贴政策。
- **季节性规律**：农产品受种植和收获季节影响明显。
- **突发事件**：自然灾害、地缘政治冲突。

> 关注方式：交易所公布的库存数据、行业协会报告、国家统计局数据等。

### 7.2 技术面分析

技术分析基于"价格反映一切信息"的假设，通过研究历史价格和成交量来预测未来走势：

**常用工具：**
- **K线图（Candlestick）**：最基础的价格图表，展示开盘价、收盘价、最高价、最低价。
- **均线（Moving Average）**：如 MA5、MA10、MA20，用于判断趋势方向。
- **成交量与持仓量**：量价配合分析，放量突破、缩量回调等。
- **支撑位与阻力位**：价格反复测试但未突破的关键价位。
- **MACD、KDJ、RSI**：常见的辅助技术指标。

### 7.3 新手常见误区

1. **重仓交易**：一次投入过多资金，一两次亏损就可能伤及本金。
2. **不设止损**：死扛亏损单，期待行情反转，往往越亏越多。
3. **频繁交易**：手续费和滑点累积，侵蚀利润。
4. **追涨杀跌**：在价格大幅波动时冲动下单，买在高点、卖在低点。
5. **只关注盈利**：忽视风险管理，赚小亏大。
6. **情绪化交易**：亏损后急于翻本，或盈利后过度自信加仓。

## 8. 新手入门建议

### 8.1 模拟交易练习

大多数期货公司和第三方软件（如文华财经、博易大师）提供模拟交易功能。新手应先在模拟环境中练习至少1~3个月：
- 熟悉交易软件操作。
- 练习下单、平仓、设置止损。
- 验证自己的交易策略是否有效。
- 体验价格波动对保证金和盈亏的影响。

### 8.2 从小资金开始

完成模拟交易后转入实盘，建议：
- 初始资金 1~3万元即可。
- 选择保证金较低的品种起步（如玉米、豆粕，一手保证金约两三千元）。
- 每次只交易1~2手。
- 把实盘初期当作"交学费"的过程，重点是积累经验而非盈利。

### 8.3 交易纪律与心态

- **制定交易计划**：每次开仓前明确入场理由、止损位、目标位。
- **严格执行止损**：止损是保护本金的底线，不可随意取消或移动。
- **保持交易记录**：记录每笔交易的开平仓理由、盈亏结果，定期复盘。
- **接受亏损**：亏损是交易的一部分，单笔亏损不代表策略失败。
- **控制交易频率**：没有明确信号时宁可空仓等待。
- **避免情绪驱动**：不因恐惧或贪婪做出非计划内的操作。

### 8.4 推荐学习资源

- **交易所官方网站**：上期所、大商所、郑商所、中金所的投资者教育栏目。
- **期货从业资格考试教材**：系统学习期货基础知识的权威材料。
- **交易软件**：文华财经（WH6/WH8）、博易大师、同花顺期货通。
- **经典书籍**：
  - 《期货市场技术分析》（约翰·墨菲）——技术分析入门经典
  - 《日本蜡烛图技术》（史蒂夫·尼森）——K线图解读
  - 《海龟交易法则》（柯蒂斯·费思）——系统化交易思维

## 9. 实战案例

### 9.1 商品期货案例：螺纹钢做多交易

**背景：** 小王是一位期货新手，账户资金5万元。他观察到近期基建政策利好，螺纹钢需求预期增加，技术面上价格突破了前期盘整区间。

**交易过程：**

| 步骤 | 操作 | 详情 |
|------|------|------|
| 1. 分析 | 确定方向 | 基本面利好 + 技术面突破，判断看涨 |
| 2. 开仓 | 买入开仓 | 3500元/吨 买入2手（合约单位10吨/手） |
| 3. 保证金 | 占用资金 | 3500 × 10 × 2 × 10% = 7,000元 |
| 4. 止损 | 设定止损 | 3450元/吨（亏损 50 × 10 × 2 = 1,000元，占总资金2%） |
| 5. 止盈 | 设定目标 | 3650元/吨（盈利 150 × 10 × 2 = 3,000元） |
| 6. 平仓 | 价格到达目标 | 3650元/吨卖出平仓 |
| 7. 结果 | 盈利 | 3,000元，收益率 3,000/7,000 ≈ 42.9% |

**要点总结：**
- 开仓前有明确的分析依据和交易计划。
- 仓位控制在总资金的14%（7,000/50,000）。
- 止损明确，最大亏损可控（占总资金2%）。
- 盈亏比 = 3,000 / 1,000 = 3:1，是合理的盈亏比。

### 9.2 套期保值案例：大豆种植户锁定售价

**背景：** 张农户种植了100吨大豆，预计3个月后收获上市。当前大豆现货价格 5,000元/吨，他担心收获时价格下跌影响收入。

**套保操作：**

| 步骤 | 操作 | 详情 |
|------|------|------|
| 1. 现状 | 持有现货头寸 | 预期3个月后有100吨大豆要出售 |
| 2. 期货操作 | 卖出开仓 | 在期货市场以5,000元/吨卖出10手大豆期货（10吨/手） |
| 3. 价格下跌 | 3个月后现货跌至4,600元/吨 | 现货亏损：(5,000-4,600)×100 = 40,000元 |
| 4. 期货平仓 | 买入平仓 | 期货价格同步跌至4,600元/吨，期货盈利：(5,000-4,600)×10×10 = 40,000元 |
| 5. 结果 | 期现对冲 | 现货亏损被期货盈利弥补，实际售价锁定在约5,000元/吨 |

**要点总结：**
- 套期保值的目标不是盈利，而是**锁定价格、规避风险**。
- 如果价格上涨，现货多卖的钱会被期货亏损抵消，同样锁定在5,000元/吨左右。
- 这正是期货市场"风险转移"功能的典型应用。

### 9.3 反面案例：忽视风险管理导致爆仓

**背景：** 李某账户资金10万元，看好原油价格上涨，决定重仓做多。

**交易过程：**

| 步骤 | 操作 | 详情 |
|------|------|------|
| 1. 开仓 | 重仓买入 | 满仓买入原油期货，占用保证金约9万元 |
| 2. 风控 | 未设止损 | 认为原油一定会涨，不需要止损 |
| 3. 行情反转 | 价格下跌5% | 亏损 = 合约价值 × 5% ≈ 45,000元 |
| 4. 追保 | 收到追保通知 | 可用资金不足，期货公司通知补足保证金 |
| 5. 侥幸心理 | 未补足保证金 | 坚信行情会反转，选择扛单 |
| 6. 强平 | 被强制平仓 | 次日开盘继续下跌，期货公司强制平仓 |
| 7. 结果 | 巨额亏损 | 亏损约6万元，账户仅剩4万元 |

**教训总结：**
- **满仓操作**是最危险的行为，没有留出任何应对波动的空间。
- **不设止损**意味着把风险完全暴露，将小亏变成了大亏。
- **侥幸心理**让本可以控制的亏损持续扩大。
- 如果李某控制仓位在30%（3万元保证金）并设置2%止损，最大亏损仅约2,000元。
